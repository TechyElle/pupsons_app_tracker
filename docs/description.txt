Harvest: Automated Member Engagement and Tracking Platform for PUP SONs

Easy Explanation

This project is a system for managing student organization members.
It allows officers to:

Register members digitally
Assign roles and committees
Track attendance automatically
Monitor active/inactive members
Generate reports
Securely manage access
Send announcements

In simple words:
It replaces manual attendance sheets and spreadsheets with an automated organization management system.
